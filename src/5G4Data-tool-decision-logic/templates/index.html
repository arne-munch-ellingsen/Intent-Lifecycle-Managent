About This Tool

This tool will decide what action that it is needed to take selecting from the possible action types. The actions to meet the required total latency are of one of these types:

  1. It is only needed to configure a network slice
  2. It is only needed to move the application to a datacenter that is closer to the handsets (UE)
  3. It is needed to do both a) and b)

For b), the local datacenter that the application must be moved to may be drawn from a set of local datacenters that would all make it possible to meet the latency requirement specified.

It could be that some of the values we are asking for in this tool could be calculated by AI agents based on the synthetic data we have provided about the infrastructure. This is something the tool owners need to dwell on.
