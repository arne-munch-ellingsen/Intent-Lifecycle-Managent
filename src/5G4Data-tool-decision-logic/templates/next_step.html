Next Step: Intent Creation

To be able to fill in network QoS Expectations in the Intent the following needs to be established:

  1. The expected latency end to end (in ms)
  2. The expected bandwidth (in mbit/s)
  3. The area the slice should be valid for (geo polygon)
  4. Other needs (e.g., green expectations, cost expectations)

To be able to fill in deployment Expectations in the Intent the following needs to be established:

  1. The Edge datacenter for deployment
  2. The compute latency requirement
  3. The workload that should be deployed (unique ID from the catalog)
  4. The workload deployment descriptor URL
